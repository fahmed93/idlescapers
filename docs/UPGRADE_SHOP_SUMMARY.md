# Upgrade Shop Summary

## Overview
The upgrade shop contains purchasable upgrades that improve skill training speed. Each upgrade belongs to a specific skill and provides a percentage speed modifier.

## Requirements Met
All 15 skills now have exactly 5 upgrades each, meeting the requirement of 5-8 upgrades per skill.

## Statistics
- **Total upgrades**: 75 across 15 skills
- **All skills have exactly 5 upgrades** (within the 5-8 requirement)
- **Level range**: 1 to 95
- **Cost range**: 100 to 45,000 gold
- **Speed modifier range**: +10% to +80%

## Design Principles
1. **Progressive Unlocking**: Upgrades unlock at increasing level requirements
2. **Escalating Costs**: Higher tier upgrades cost significantly more gold
3. **Increasing Benefits**: Speed modifiers increase with each tier
4. **Thematic Naming**: Each skill has thematically appropriate upgrade names
5. **Balanced Progression**: Similar patterns across all skills for consistency

## Usage
Upgrades are purchased through the UpgradeShop autoload:
```gdscript
# Purchase an upgrade
UpgradeShop.purchase_upgrade("basic_fishing_rod")

# Get speed modifier for a skill
var fishing_speed = UpgradeShop.get_skill_speed_modifier("fishing")

# Get all upgrades for a skill
var fishing_upgrades = UpgradeShop.get_upgrades_for_skill("fishing")
```
