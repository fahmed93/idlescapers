# Game Name Brainstorming Report
## IdleScapers - Official Name Alternatives Analysis

**Date**: December 25, 2025  
**Current Name**: IdleScapers  
**Project**: RuneScape/Melvor Idle-inspired idle progression game

---

## Executive Summary

This report presents 15 carefully selected game name alternatives for the project currently known as "IdleScapers." Each name has been analyzed across multiple dimensions including:
- **Brand Identity**: How well it represents the game's core mechanics
- **Market Appeal**: Target audience resonance and memorability
- **SEO Performance**: Search engine discoverability and uniqueness
- **Legal Risk**: Trademark concerns and similarity to existing games
- **Domain Availability**: Web presence potential

### Quick Recommendations
- **Best Overall**: SkillForge Idle (Strong branding, clear mechanics, SEO-friendly)
- **Most Unique**: Incremental Ascent (Distinctive, low competition)
- **Best for Marketing**: Epic Idle Quest (Broad appeal, easy to understand)
- **Keep Current**: IdleScapers (Already established, decent SEO, clear meaning)

### 2. IdleScapers (Current Name)

- **First Impression**: Clearly an idle game, but might seem derivative.

#### SEO Performance (6/10)
- **Search Competition**: Medium competition. Multiple "idle + scape" combinations exist.
- **Keyword Relevance**: Good. Contains both "Idle" and references RuneScape.
- **Discoverability**: Moderate. May get lost among similar named games.

#### Legal Risk (Medium)
- Jagex (RuneScape owner) has been protective of "Scape" in game names
- Multiple similar games exist without issue, but risk exists
- Recommend avoiding if planning commercial release

#### Domain Availability
- idlescapers.com: Check availability
- idlescapers.io: Already using GitHub.io subdomain

#### Overall Score: 6.7/10
**Recommendation**: Acceptable current name, but has room for improvement. Consider rebranding before major launch to avoid potential legal issues and improve discoverability.

## Conclusion

The current name **IdleScapers** is serviceable but has room for improvement, particularly regarding SEO competitiveness and potential legal concerns with the "Scape" suffix. 

**SkillForge Idle** emerges as the strongest alternative, offering significant improvements in brand identity, search engine optimization, and market differentiation while maintaining clear genre communication.

However, if the project has already built some recognition under IdleScapers, the cost of rebranding may outweigh the benefits. In that case, focusing development resources on game quality and marketing would be more valuable than a name change.

For a project in early development with no established brand recognition, **SkillForge Idle** is the recommended path forward.
