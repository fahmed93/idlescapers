## Benefits of This Display

1. **Non-intrusive**: Fits naturally between the current action time and stop button
2. **Clear hierarchy**: Each piece of information has its own line
3. **Color-coded**: Light blue distinguishes it from other time displays
4. **Contextual**: Only shows when actively training
5. **Informative**: Provides valuable planning information at a glance

## Mobile Layout Considerations

The training panel is part of the main content area which has:
- Mobile viewport: 720x1280 pixels
- Sidebar: ~120px (collapsible)
- Content area: ~560px wide

The time to level label:
- Font size: 12 (same as action time)
- Centered alignment
- Adequate touch target spacing around stop button
- Readable without zooming on mobile devices
